Experiment Instructions

Purpose of the Study

This experiment aims to evaluate the quality of reconstructed virtual objects generated by various image-based 3D object reconstruction techniques. Your feedback will provide valuable information for developing better reconstruction algorithms.

How the Experiment Works

On each screen, you will see the following elements:

  • Left: Reference images - These show the actual object for reference. You can click each image to see the details.
  • Center: Rendered images to evaluate - You can move the slider to change frames.
  • Right: Questions - Please answer the quality assessment questions for each rendered images.

You will first go through a brief practice session to get familiar with the interface before moving on to the actual experiment.

Before You Begin

The experiment takes approximately 60 minutes to complete.
